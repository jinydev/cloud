# ☁️ Cloud & DevOps 마스터 코스

본 커리큘럼은 온프레미스 리눅스(Linux) 환경에서 다져진 핵심 인프라 역량을 바탕으로, **아마존 웹 서비스(AWS) 기반의 클라우드 컴퓨팅 환경과 도커(Docker), 쿠버네티스(K8s) 등의 모던 컨테이너 오케스트레이션** 기술을 내재화하기 위해 기획된 14주차 심화 확장 코스입니다.

수동으로 제어하던 모든 리소스를 코드로 관리(IaC)하고 빌드부터 배포까지의 자동화 파이프라인(CI/CD)을 구축하는 현업 데브옵스(DevOps) 엔지니어의 여정을 밟게 됩니다.

---

## 📚 14주차 클라우드/데브옵스 커리큘럼 (Curriculum)

가장 빈틈없는 학습을 위해 4개의 거대한 파트(Part 1~4)로 전면 재설계 되었습니다. 각 주차별 개념과 실습을 순차적으로 정복해 보십시오.

* **[5장. 컴퓨팅 인스턴스 론칭 (EC2)](/05_compute/)**
